# Team VR Studio V1

Standalone GitHub Pages package for the earlier team VR Studio view.

Published URL:

https://neurobitengineer.github.io/vr-studio/

This page is a focused copy of the Cloudflare-backed VR Studio. It displays only:

- Living Room
- Bathroom
- Dining Room
- Kitchen
- Bedroom
- Classroom

The underlying Cloudflare room store is still used for loading room assets. The `env_hdri` catalog entry is intentionally not shown in the sidebar because it is an environment/HDRI asset, not a room option for this V1 deliverable page.
